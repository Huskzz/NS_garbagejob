# NS_garbagejob
A high-performance, immersive garbage collection job for ESX Framework, built using ox_lib and ox_target.

Features:
- Dynamic Dumpsters: Each dumpster has a random capacity of 1–5 bags.

- Truck Capacity: Real-world tracking; the truck holds a maximum of 100 bags.

- Immersive Interactions: Includes custom animations, physical trash bag props, and ox_target interaction zones.

- Framework Agnostic Notifications: Easily toggle between esx, ox_lib, okokNotify, or okokTextUI.

- Secure: Server-side payout verification to prevent exploitation.

Dependencies:
- ESX

- ox_lib

- ox_target

Integrated:
okokNotify
